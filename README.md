## 첫번째 양파 껍질 벗기기 

문자열 계산기를 짜보자!

## 요구사항 

* 전달하는 문자를 구분자로 분리 후 각 숫자의 합을 구한다
* 쉼표,콜론을 구분자로 가지는 문자열을 전달하는 경우 구분자를 기준으로 분리 한 후 각 숫자의 합을 반환한다
* 커스텀 구분자를 지정 할 수 있는데, `//`과 `\ㅜ` 사이에 위치하는 문자를 커스텀 구분자로 사용한다.
* 문자열 계산기에 음수를 전달하는 경우 `RuntimeException`으로 예외 처리 해야 한다.
